## AI Characters

This repo is a collection of AI character cards in PNG and JSON format.

### Roleplay

The [characters](characters/) directory contains cards intended for interactive roleplaying in a chat format. These are all SFW and suitable for most models.

### Utilities

The [utilities](utilities/) directory contains cards that perform some useful fucnction, such as translating one format of input into another.
